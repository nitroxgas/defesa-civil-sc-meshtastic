# Contribuindo

Contribuições são bem-vindas, especialmente para:

- versão standalone sem Home Assistant;
- envio direto pela biblioteca Python do Meshtastic;
- envio via MQTT;
- suporte a outras Defesas Civis estaduais;
- parser CAP/Common Alerting Protocol;
- filtros por município/região;
- testes unitários para compactação de mensagens.

Antes de abrir pull request, remova dados reais de gateway, canais, chaves, logs e arquivos de estado.
